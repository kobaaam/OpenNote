# Themes
OpenNote currently has two supported themes

Light:
![][light]
![][lightEdit]

and Dark:
![][dark]
![][darkEdit]

to flip between them, simple change the following line in `/OpenNote/Config.php`
```php
	public static $theme = "dark"; //Default: light; //"dark" also available
```
